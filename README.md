Deskripsi Aplikasi
MyTaskList adalah aplikasi manajemen tugas interaktif yang dirancang untuk membantu pengguna mengorganisir dan melacak aktivitas harian mereka secara efisien. Dibangun menggunakan React Native dengan antarmuka dark mode yang modern, aplikasi ini memudahkan pengguna untuk mencatat tugas, mengatur tingkat prioritas, serta menyaring daftar pekerjaan berdasarkan status penyelesaiannya.
